# 📊 Customer Churn Prediction using Machine Learning

Customer churn is a major concern for businesses, especially in industries like telecom, banking, and SaaS. Losing customers directly impacts revenue and increases customer acquisition costs.  

This project focuses on building a Machine Learning model to predict whether a customer will leave the company (Churn: Yes/No) based on behavioral and demographic data.

The goal is not only to build an accurate model but also to generate actionable insights that help businesses improve customer retention.

---

## 🚀 Project Overview

This project follows a complete end-to-end Machine Learning pipeline:

- Data preprocessing (handling missing values, duplicates, and outliers)
- Exploratory Data Analysis (EDA)
- Feature engineering, encoding, and scaling
- Handling class imbalance using SMOTE
- Training multiple classification models:
  - Logistic Regression  
  - K-Nearest Neighbors (KNN)  
  - Decision Tree Classifier  
  - Random Forest Classifier  
  - Support Vector Classifier (SVC)  
  - Gradient Boosting Classifier  
  - XGBoost Classifier  
- Model evaluation using:
  - Accuracy  
  - Precision  
  - Recall  
  - F1 Score  
  - ROC-AUC Score  
  - Confusion Matrix  
- Hyperparameter tuning using GridSearchCV and RandomizedSearchCV  
- Model comparison and selection of the best-performing model  

---

## 🎯 Objective

- Predict customer churn using historical data  
- Identify key factors influencing customer behavior  
- Provide data-driven strategies to reduce churn  

---

## 📊 Key Insights

- High monthly charges are strongly associated with higher churn  
- Customers with month-to-month contracts are more likely to leave  
- Lack of tech support increases churn probability  
- Customers with shorter tenure show higher churn rates

This project demonstrates how Machine Learning can be leveraged to:
